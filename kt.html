Knight's Tour Puzzle Solver

This software is a JavaFx application, designed and implemented with multi-thread graph search algorithms for solving the ancient Knight's tour puzzle. A deployed desktop stand-alone version is available for Mac OS X. Download here (.dmg file). Source codes are here, and you can also run it from the command line.

Furthermore, owing to the extremely large quantity of computation needed for finding the knight's tour, some of the starting position might take around 2 minutes to receive results. Based on my test on a standard MacBook Air, by using this software, most of the solutions can be obtained within several seconds. Good luck and enjoy!
